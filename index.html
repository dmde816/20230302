이틀전까지 한 내용은 태그 : div, p, h1~h6, span, br, img, table, a, 스타일을 줘보자 속성(attribute) style을 주자
style 속성 = css(cascading style sheet)
width: 너비, max-width, min-width
height : 높이, max-height, min-height
background-color : 배경색, 색상을 태그에 입힐 수 있다., tgb(255,255,255); padding(태그 내부 여백), margin(외부의 여백) : top, right, bottom, left 값을 한 개만 썼을 때는 사방면이 그 값으로 적용된다. 각각 지정하고 싶을 때는 4개 값 모두 작성 해줘도 된다. ex top 100, bottom 100 ... 값을 두개 작성했을 때 상하 좌우로 짝을 맞춰 지정해야함. box-sizing : 박스의 크기를 어떻게 설정할지 기준을 설정
속성을 추가할 때 속성명= "속성 값" diplay: 영역의 노출 크기를 어떻게 설정할지 기준을 설정한다. 부모 태그를 기준으로 모든 너비를 차지한다. 높이는 자식의 기준으로 body가 부모이면 바디만큼의 너비를 가지고, div를 부모로 쓰고 부모 div의 크기를 제한하면 그만큼의 크기를 가진다. inline-block 본인의 자리만 차지함
